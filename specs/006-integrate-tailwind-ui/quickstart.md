# Quickstart

No new dependencies or database migrations are introduced in this phase.

To test the UI updates locally:
1. Ensure the application is running (e.g., via XAMPP or PHP built-in server `php -S localhost:8000 -t .` in the root).
2. Log in sequentially as HR Admin, Candidate, and Technical Interviewer.
3. Verify that the role-based navigation logic functions as intended with the new Tailwind-based layout.
4. Resize the browser to mobile width and verify responsiveness.
