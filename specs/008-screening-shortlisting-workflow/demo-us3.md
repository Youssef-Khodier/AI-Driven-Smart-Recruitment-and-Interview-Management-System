# Demo US3: Automated Triage From Applied

1. Ensure US2 has been completed.
2. Login as an HR Admin.
3. Navigate to HR Dashboard -> Requisitions.
4. Open the configured requisition with APPLIED applications.
5. Go to "View Shortlist".
6. Click "Run Triage".
7. Verify the preview table correctly maps candidates' scores to target statuses.
8. Click "Confirm Triage".
9. Verify the results summary page correctly shows counts per status.
10. Return to the Shortlist and verify applications have been moved from APPLIED to the new status.
